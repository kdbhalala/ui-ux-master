## [1.0.0] - 2026-01-30

### Added
- Initial release of UI/UX Master skill
- Complete Apple HIG coverage (iOS, macOS, watchOS, tvOS, visionOS)
- Modern web design patterns and best practices
- SuperDesign integration (oklch colors, glassmorphism, neo-brutalism)
- Comprehensive color systems guide
- Typography and font pairing guidelines
- Layout patterns (CSS Grid, Flexbox, 8pt grid)
- Component design library (buttons, forms, cards, navigation)
- Animation and motion principles
- Dark mode implementation guide
- WCAG AA/AAA accessibility guidelines
- Responsive design patterns
- 16 reference files for deep dives
- Code examples (SwiftUI, CSS, HTML)
- Design checklists and decision guides
- Tool recommendations

### Documentation
- README.md with installation and usage instructions

- CHANGELOG.md (this file)
- Comprehensive SKILL.md main guide
